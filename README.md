# dotfiles

The bootstrap scripts is assumed on running with OS-X.
Only tested with macOS Mojave (version 10.14).

## Install

```
bash -c "$(curl -L https://raw.githubusercontent.com/aikuma0130/dotfiles/master/bootstrap.sh)"
```

## After installation

### nvim init

* execute neovim
```
nvim
```

* Run below command
```
:UpdateRemotePlugins
```
